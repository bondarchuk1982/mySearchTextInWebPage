Ответы на дополнительные вопросы в задании:
1.
- Было принято решение, что закачка и поиска по страницам будет происходить в отдельных потоках. Общее количество активных потоков не должно превышать заданный максимум.
- Было принятро решение, что максимальное количество сканируемых url это максимальное число страниц, которые будут обработаны в общей сумме.

2. С помощью Google testing framework (gtest). Фреймворк предоставляющий большой набор инструментов для автотестирования. На мой взгляд обычно методы тестируются так:
- сначала по принцыпу выбора крайних углов.
- потом стресовые состояния
- некорректные параметы
- если передаваемые параметны не должны изменяться то проверка на их сохранность
- и в конце уже обычные состояния.
